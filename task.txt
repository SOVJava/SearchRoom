Поиск номера в Airbnb

Введение
Вам предоставляется два файла CSV, которые содержат информацию о комнатах Airbnb, предлагаемых посетителям
Нью-Йорка, США. Каждая комната принадлежит определенному соседству, соседство, в свою очередь, связано с
одним из этих районов: Статен-Айленд, Куинс, Бронкс, Бруклин и Манхэттен. Основная информация о номерах
хранится в файле 'rooms.csv'. Связи между районами и районами хранятся в файле 'boroughs.csv'.

Описание задания
Создайте новое консольное приложение, добавьте исходные файлы CSV в проект. Реализуйте следующие задачи:
1. Разработайте объектно-ориентированную модель данной предметной области со следующими объектами:
    Комната (идентификатор, имя, цена, минимальное количество ночей, количество отзывов),
    Хост (идентификатор, имя), Соседство (имя, название района).
2. Загрузите весь набор данных (из обоих файлов) в память программы. В конце вам нужно иметь несколько
    коллекций (комнаты, хосты, окрестности и, возможно, районы) разных размеров (например, количество
    хостов меньше, чем количество комнат, так как один хост может предоставить несколько комнат)
    с объектами должным образом связаны друг с другом (комната логически связана с хостом и соседством).
3. Рассчитайте среднюю цену среди всех комнат в каждом районе. Распечатайте результат на экране.
4. Реализовать повторяющуюся работу по поиску комнаты для пользователя. При запуске поиска комнаты
    пользователь вводит следующие параметры:
        1 название города
        2 типа номера (один из 3 вариантов: отдельная комната, общая комната и весь дом / квартира)
        3 количество ночей (программа должна фильтровать номера, чтобы минимальное количество ночей
            было меньше или равно этому значению)

    Программа должна найти соответствующие номера и распечатать их в порядке возрастания общей стоимости
(цена за одну ночь из таблицы, умноженная на количество ночей, введенных пользователем). Для каждой
комнаты, которая соответствует критериям поиска пользователя, выведите следующую информацию:
    Имя
    Имя хоста
    Цена за ночь
    Общая стоимость

5. Добавьте необходимую защиту от возможных ошибок (файлы отсутствуют, неверный формат файла, неправильный
    ввод данных пользователем).
